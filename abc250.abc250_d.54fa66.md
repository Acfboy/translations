### 题意 

定义一个正整数 $k$ 为 "250数"，满足 $k=p\times q^3$ ，其中 $p,q$ 为质数并且 $p < q$ 。

现给出 $N$ 求有多少个小于等于 $N$ 的 "250数" 。

### 数据范围

$1\le N\le 10^{18}$



---

### 输入格式

一个数 $N$ 。

### 输出格式

输出一个数表示答案。

---

### 样例输入1

```
250
```

### 样例输出1

```
2
```

### 样例解释1

$54=2\times 3^3$，是 "250数" 。

$250=2\times 5^3$，是 "250数" 。



---

### 样例输入2

```
1
```

### 样例输出2

```
0
```

---

### 样例输入3

```
123456789012345
```

### 样例输出3

```
226863
```
