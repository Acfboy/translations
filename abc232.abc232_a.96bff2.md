## 题意

给你一个长度为 $3$ 的字符串，其由两个 $1 \sim 9$ 的整数 $a,b$ 和一个字符 `x` 按 `axb` 的顺序组成。请你输出 $a \times b$ 的结果。

## 数据范围

- 字符串长度为 $3$。
- 字符串的第 $1$ 和 $3$ 个字符均为 $1 \sim 9$ 的整数。
- 字符串的第 $2$ 个字符为 `x`。

## 输入格式

一行一个字符串。

## 输出格式

输出一行一个数表示答案。

## 样例

### 样例输入1

```input
3x7

```

### 样例输出1

```output
21

```

### 样例解释1

$3 \times 7 = 21$

### 样例输入2

```input
9x9

```

### 样例输出2

```output
81

```

### 样例输入3

```input
1x1

```

### 样例输出3

```output
1

```
