## 题意  

有 $N$ 座城市，有 $M$ 条连接 $A_i,B_i$ 的单向道路。     
求有多少对城市 $(X,Y)$ 使得能从 $X$  到达 $Y$。

## 数据范围

$1\le N\le 2000$，         
$1\le M\le 2000$，             
$1\le A_i,B_i\le N$，            
$A_i \neq B_i$，      
给出的边没有重边，        
所有输入的数都是整数。  

## 输入格式

第一行两个整数 $N,M$。        
第二行到第 $M+1$ 行每行两个整数，$A_i$ 和 $B_i$。
          
## 输出格式

一行，一个整数，合法的城市对数。

## 样例

### 输入1 :
```
3 3
1 2
2 3
3 2
```

### 输出1 :
```
7
```

### 输入2 :
```
3 0
```

### 输出2 :
```
3
```

### 输入3 :
```
4 4
1 2
2 3
3 4
4 1
```

### 输出3 :
```
16
```

## 样例解释

对于样例一，合法的城市对有 $(1,1),(1,2),(1,3),(2,2),(2,3),(3,2),(3,3)$。
