## 题意

输入一个整数 $N$，你需要进行 $K$ 次操作。          
- 如果 $N$ 能整除 $200$，那么将 $N$ 除以 $200$。     
- 否则，将 $N$ 变为 $N\times 1000+200$。      

输出经过 $K$ 次操作后的 $N$。     

## 数据范围

$1\le N\le 10^5$，        
$1\le K\le 20$。

## 输入格式

一行两个整数 $N,K$。

## 输出格式

一行一个整数表示 $K$ 次操作后的 $N$。

## 样例 

### 输入1 :
```
2021 4
```   

### 输出1 :
```
50531
```  

### 输入2 :
```
40000 2
``` 

### 输出 2:
```
1
```

### 输入3 :
```
8691 20
```

### 输出3 :
```
84875488281
```

## 样例解释

对于样例 1，$2021\to 2021200 \to 10106 \to 10106200 \to 50531$。
