## 题意

输入一个长度为 $N$ 的序列 $A$，你可以进行如下操作：       

- 选择一个 $x$，选择一个 $y$，将序列中所有的 $x$ 都替换成 $y$。

你需要求出最少需要几次操作使 $A$ 变为回文的。

## 数据范围

$1\le N,A_i\le 2\times 10^5$，

所有输入的都是整数。

## 输入格式

第一行一个整数 $N$。      

接下来 $N$ 个整数 $A_i$。

## 输出格式

一行一个整数，最少的操作次数。 

## 样例

### 样例输入1

```
8
1 5 3 2 5 2 3 1
```



### 样例输出1

```
2
```



### 样例输入2

```
7
1 2 3 4 1 2 3
```



### 样例输出2

```
1
```

### 样例输入 3

```
1
200000
```



### 样例输出 3

```
0
```



### 样例解释

对于样例 1，

第一次操作选择 $(3,2)$，

操作完之后的序列是 $(1,5,2,2,5,2,2,1)$。

第二次操作选择$(2,5)$，

操作完之后的序列是 $(1,5,5,5,5,5,5,1)$。

对于样例 2，

开始时已经回文了。
