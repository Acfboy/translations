## 题意

高桥一开始穿白色衣服，穿白衣服的下一天他会穿黑衣服，同理，穿黑衣服的下一天他会穿白衣服。

问 $N$ 天后他穿什么颜色的衣服。

## 数据范围

$1\le N\le 30$
## 输入格式

读入一个 $N$。

## 输出格式

输出他 $N$ 天后穿着衣服的颜色，黑色为 `Black`，白色为 `White`。

### 样例输入1
```
2
```
### 样例输出1
```
White
```
### 样例输入2
```
5
```
### 样例输出1
```
Black
```
