## 题意

有多少个不大于 $N$ 的正整数，使得该正整数各位成绩不大于 $K$。

## 数据范围

$1\le N\le 10^{18}$。

$1\le K\le 10^9$。

输入的都是整数。

## 输入格式

一行两个整数 $N,K$。

### 输出格式

一行，一个整数，满足条件的正整数数量。

## 样例

### 样例输入 1

```
13 2
```

### 样例输出 1

```
5
```

### 样例输入 2

```
100 80
```

### 样例输出 2

```
99
```

### 样例输入 3

```
1000000000000000000 1000000000
```

### 样例输出 3

```
841103275147365677
```

## 样例解释

对于样例 1，合法的正整数有 $1$，$2$，$10$，$11$，$12$，共 $5$ 个。
