## 题意

给出一个字符 $\alpha$，若其为大写字母则输出 `A`，反之则输出 `a`。

## 数据范围

- $\alpha$ 为大写或小写英文字母。

## 输入格式

第一行输入一个字符 $\alpha$，含义如题所示。

## 输出格式

输出一行为答案。

## 样例

### 样例输入1

```
B
```

### 样例输出1

```
A
```

### 样例解释1

`B` 为大写字母，所以输出 `A`。

### 样例输入2

```
a
```

### 样例输出2

```
a
```

### 样例解释2

`a` 为小写字母，所以输出 `a`。
