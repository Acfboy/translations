### 题意 
求大于等于 $x$ 的第一个质数。
### 数据范围
$x\le10^5$

---
### 输入格式
一行 $1$ 个整数 $x$。
### 输出格式
一行 $1$ 个整数表示大于等于 $x$ 的第一个质数。

---
### 样例输入1
```
20
```
### 样例输出1
```
23
```
### 样例解释1
大于等于 $20$ 的第一个整数是 $23$

---
### 样例输入2
```
2
```
### 样例输出2
```
2
```

---
### 样例输入3
```
99992
```
### 样例输出3
```
100003
```
