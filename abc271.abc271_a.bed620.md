## 题意

给定十进制整数 $N$，输出其十六进制表示，其中字母输出用大写字母。

## 数据范围

- $0\le N\le 255$。

## 输入格式

一行一个整数 $N$。

## 输出格式

一行，$N$ 的十六进制表示。输出两位，不足的前导零补足。

## 样例

### 样例输入1

```
99
```

### 样例输出1

```
63
```

### 样例输入2

```
12
```

### 样例输出2

```
0C
```

### 样例输入3

```
0
```

### 样例输出3

```
00
```
### 样例输入4

```
255
```

### 样例输出4

```
FF
```
