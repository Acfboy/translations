### 题意

给一个数 $N$，求 $2^N$。

### 数据范围

- $0\leq N \leq 30$
- $N$ 是整数。

---

### 输入格式

```
N
```

### 输出格式

一行一个数 $2^N$。

---

### 样例输入1

```
3
```

### 样例输出1

```
8
```

### 样例解释1

$2^3=8$。

---

### 样例输入2

```
30
```

### 样例输出2

```
1073741824
```
