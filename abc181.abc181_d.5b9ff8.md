## 题意
给你一个字符串 $S$，问重排后能不能变成 $8$ 的倍数。
## 数据范围
$1\le |S|\le 2\times 10^5$
## 输入格式
一行一个数字串 $S$。
## 输出格式
是否能变成 $8$ 的倍数，输出 `Yes` 或者 `No`。
### 样例输入1
```
1234
```
### 样例输出1
```
Yes
```
### 样例解释1
$1432$
### 样例输入2
```
1333
```
### 样例输出1
```
No
```
### 样例输入2
```
8
```
### 样例输出1
```
Yes
```
