### 题意 
给定字符串 $S$ 和 $T$，输出 $T+S$
### 数据范围
$|S,T|\le100$

---
### 输入格式
一行两个字符串 $S$,$T$
### 输出格式
一行一个字符串 $T+S$

---

### 样例输入1
```
oder atc
```
### 样例输出1
```
atcoder
```
### 样例解释1
$S$=`oder` $T$=`atc`

---
### 样例输入2
```
humu humu
```
### 样例输出2
```
humuhumu
```
