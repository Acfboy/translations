### 题意

给一个由大写英文字母组成的长度为 $4$ 的字符串 $S$。

判断 $S$ 是否正好由两个各正好出现两次的不同的字母组成。

### 数据范围

$|S|=4,S$ 由大写字母组成。

---

### 输入格式

```
S
```

### 输出格式

若 $S$ 是否正好由两个各正好出现两次的不同的字母组成，输出  `Yes`，否则输出 `No`。 

---

### 样例输入1

```
ASSA
```

### 样例输出1

```
Yes
```

### 样例解释1

$S$ 由 $A$ 和 $S$ 组成，$A$ 和 $S$ 各出现两次。

---

### 样例输入2

```
STOP
```

### 样例输出2

```
No
```

---

### 样例输入3

```
FFEE
```

### 样例输出3

```
Yes
```

---

### 样例输入4

```
FREE
```

### 样例输出4

```
No
```
