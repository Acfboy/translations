## 题意
求出在 $1$ 到 $N$ 之间的所有不是 $A$ 或 $B$ 的倍数的整数之和。

## 数据范围

- $1 \le N,A,B \le 10^9$
- 所有读入的数值都是整数。

## 输入格式

输入是由标准输入提供的，格式如下：

```
N A B
```

## 输出格式

输出一行答案。

## 样例

### 样例一

```
10 3 5
```

```
22
```

$1,2,4,7,8$ 五个整数满足该条件，所以输出 $1+2+4+7+8=22$。

### 样例二

```
1000000000 314 159
```

```
495273003954006262
```
