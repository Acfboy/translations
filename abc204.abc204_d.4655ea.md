## 题意  

有 $N$ 道菜，每道菜需要在炉子里加热 $T_i$ 分钟。     
现在有两个炉子，但是每个炉子只能容下一碗菜（只能同时加热一个菜）。    
问最少花费的时间，使得所有菜都被加热过。              

## 数据范围
                     
$1\le N\le 100$。      
$1\le T_i\le 10^3$。      

## 输入格式

第一行一个数 $N$。     
第二行 $N$ 个整数 $T_i$。                    
          
## 输出格式

一行一个整数，最少的时间。               

## 样例

### 输入1 :
```
5
8 3 7 2 5
```

### 输出1 :
```
13
```

### 输入2 :
```
2
1000 1
```

### 输出2 :
```
1000
```

### 输入3 :
```
9
3 14 15 9 26 5 35 89 79
```

### 输出3 :
```
138
```


## 样例解释

对于样例 1，炉子 1 依次加热菜肴 5 和 1，炉子 2 一次加热菜肴 2，4，3。
