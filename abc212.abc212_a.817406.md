## 题意

Takahashi 熔化并混合了 $A$ 克纯金，$B$ 克纯银。
问最后凝固后是纯金还是纯银还是金银混合物。

## 数据范围

$0\le A,B\le 100$。
$1\le A+B$。
所有输入的都是整数。

## 输入格式

一行两个整数 $A,B$。
          
## 输出格式

一行 `Gold` 表示纯金， `Silver` 表示纯银，`Alloy` 表示混合物。

## 样例

### 输入1 :
```
50 50
```

### 输出1 :
```
Alloy
```

### 输入2 :
```
100 0
```

### 输出2 :
```
Gold
```

### 输入3 :
```
0 100
```

### 输出3 :
```
Silver
```

### 输入4 :
```
100 2
```

### 输出4 :
```
Alloy
```
