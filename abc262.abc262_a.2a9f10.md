## 题目描述

世界杯会在每个模 $4$ 余 $2$ 的年份的六月举行。

现在是 $Y$ 年的一月，问下次世界杯在哪年举行。

## 数据范围

- $2000\le Y\le 3000$

- $Y\in \mathbb Z$

## 样例

### 样例一

输入：

```
2022
```

输出：

```
2022
```

解释：

$2022\bmod 4 = 2$，所以世界杯会在 $2022$ 年举行。

### 样例二

输入：

```
2023
```

输出：

```
2026
```

### 样例三

输入：

```
3000
```

输出：

```
3002
```
