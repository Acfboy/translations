### 题意

给你一个实数 $X.Y$ ，其中 $Y$ 是一个数码。

请你按如下格式输出：

- 若 $0 \le Y \le 2$ ，输出 $X-$ ；
- 若 $3 \le Y \le 6$ ，输出 $X$ ；
- 若 $7 \le Y \le 9$ ，输出 $X+$ 。

### 数据范围

- $1 \le X \le 15$
- $0 \le Y \le 9$
- $X$ 和 $Y$ 都是整数。

---

### 输入格式

输入一个实数 $X.Y$ 。

### 输出格式

输出答案。

---

### 样例输入1

```
15.8
```



### 样例输出1

```
15+
```



---

### 样例输入2

```
1.0
```



### 样例输出2

```
1-
```



---

### 样例输入3

```
12.5
```



### 样例输出3

```
12
```
